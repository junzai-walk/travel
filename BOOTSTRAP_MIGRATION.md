## 技术特点

### Bootstrap 组件使用
- `container` / `container-fluid`: 响应式容器
- `row` / `col-*`: 网格系统
- `card`: 卡片组件
- `navbar`: 导航栏
- `btn`: 按钮样式
- `badge`: 标签组件
- `alert`: 提示框

### 响应式工具类
- `d-*`: 显示控制
- `text-*`: 文本对齐
- `mb-*`, `mt-*`, `py-*`: 间距控制
- `col-lg-*`, `col-md-*`: 响应式列宽

### 保留的自定义样式
- 品牌渐变色背景
- 时间轴可视化效果
- 卡片悬停动画
- 必尝美食特殊标识
